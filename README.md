# Think Unlimited - Digital Creative Agency Landing Page

A highly visual, modern landing page built for creative agencies, studios, and visual branding professionals. Designed with a dark-mode aesthetic, bold typography, and seamless layout transitions, this project serves as a comprehensive portfolio and conversion tool.

## 🚀 Tech Stack

* **Framework:** React
* **Styling:** Tailwind CSS (Custom themes, responsive grids, and typography)
* **UI Components:** Material UI (MUI Grid, Paper)
* **Carousels/Sliders:** React-Slick
* **Icons:** React Icons

## 🧩 Key Components

* `Header.jsx` - Responsive top navigation with mobile hamburger menu.
* `HeroSection.jsx` - Eye-catching hero banner emphasizing modern solutions.
* `OurServices.jsx` & `WorkFlow.jsx` - Interactive accordions and detailed service listings.
* `AboutSection.jsx` - MUI-powered grid displaying the agency's visual portfolio.
* `ClientAbout.jsx` - React-Slick carousel featuring client reviews and a team member photo collage.
* `NewArticle.jsx` - Asymmetric CSS grid layout for the blog section.
* `FooterSection.jsx` - Multi-column footer with subscription input.

## 🛠️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/hassanalitahir71/Landing-page.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd your-repo-name
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    # or
    yarn start
    ```

## 📂 Folder Structure
Ensure your public or `src/assets` folder contains all the necessary imagery referenced in the components (e.g., `/images/logo.png`, `/images/Article1.jpg`, etc.) for the UI to render correctly.
